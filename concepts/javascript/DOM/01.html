What is the DOM?

The Document Object Model (DOM) is a programming interface for HTML and XML documents. It represents the structure of a document as a tree-like hierarchy of objects. The DOM provides methods and properties to interact with the document and the elements in the document.

DOM Methods

  • getElementById
  • getElementsByTagName
  • getElementsByClassName
  • querySelector
  • querySelectorAll
  • createElement
  • appendChild
  • insertBefore
  • removeChild
  • replaceChild
